はりぼてキット (画像置換)

はりぼてキットは、画像表示がオフの環境にも対応できる画像置換のパーツモジュールです。

.haribotekit (別名 .imagereplace も可)と、その内側に配置される空span要素 .haribotekit .panel とで構成されます。.haribotekit内のテキストを .panel に指定した背景画像が覆い隠すことで実現する画像置換のため、画像に透過GIFや半透過PNGは使わないでください。

.haribotekit に、置換画像の高さ(width)と幅(height) を、.panel に、背景画像として置換画像のパス(background-image) を付加して使用します。

.panel には、必要に応じて background-position や background-repeat を指定することもできます。

次に示すのは、ハイパーリンクを画像で置き換える実装例です。

a.cont_sample{ width:120px; height:80px; } a.cont_sample span.panel{ background-image:url("sampleimage.gif"); }

	a.cont_sample{
		width:120px; height:80px;
	}
