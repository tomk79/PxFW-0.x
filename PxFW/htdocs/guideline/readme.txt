これは、Pickles CSS のコーディングレギュレーション兼サンプルコード集です。

【使い方】

・guideline フォルダを、コンテンツディレクトリの直下に配置する。
・guideline.csv を、サイトマップディレクトリの直下に配置する。

その後、ページID[guideline]にアクセスしてください。
