コンテンツのコーディングにおいて、最も重要なことは、『テーマとコンテンツの役割の分担』です。1つのHTMLの中で役割を分担するわけですから、互いに意図せぬ効果を波及させてしまわないために、両者の間に約束事が必要になります。

以下に定義する基本規則を守って運用してください。

基本ルール
Pickles CSS の構成や運用する上での規則についての概要をまとめています。はじめにこの項目を読んでください。

次の項目は、Pickles CSS による、コンテンツのコーディングに利用できるモジュールの一覧とサンプルソース集です。

スタティックモジュール
スタティックモジュールは、テーマやメディアタイプによらず、常に固定の効果を提供します。主にショートハンド的なモジュールが含まれます。
パーツモジュール
パーツモジュールは、部品を提供します。他のパーツと組み合わせて使用できます。
ユニットモジュール
ユニットモジュールは、ユニット単位のコーディング規則を定義します。ユニットは、水平貫通の矩形ブロックで、上下に適切なマージンを与えられた概念です。
