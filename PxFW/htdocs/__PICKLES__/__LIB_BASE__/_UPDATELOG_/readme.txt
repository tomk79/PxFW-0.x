
---------- このディレクトリについて ----------

このディレクトリには、Pickles Framework Install Toolが、
Pickles Framework のベース層ファイルのインストールや
更新を行った履歴を保存する領域として確保されました。
Pickles Framework のシステムには影響しません。
また、アプリケーションから参照しないでください。

ここには、ベース層の各ファイルの更新履歴のみが蓄積され、
その他の、例えばパッケージ層ファイルの更新や、
プロジェクトスケルトンのセットアップなどの履歴は含まれません。

Tomoya Koyanagi.
